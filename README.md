# Books service
A RESTful API for adding, storing, reading and deleting records about books.
## Installation
```commandline
pip install -r requirements.txt
```
## Running the server
```
uvicorn main:app
```
## How to use
You can visit the documentation [here](http://127.0.0.1:8000/docs) (the server should be running).
## Running tests
```
pytest
```
